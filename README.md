# Solana Sentinel Node 🐋

A professional, mobile-first Python suite designed for real-time monitoring of the Solana Mainnet. This project demonstrates high-performance blockchain interaction and secure DevOps practices within a **Termux (Android Linux)** environment.

 Overview
The **Solana Sentinel** provides a lightweight bridge between mobile infrastructure and global blockchain data. It is architected to handle network latency and secure API management, making it ideal for 24/7 background monitoring.

 Technical Stack
- **Language:** Python 3.x
- **Infrastructure:** QuickNode RPC (Mainnet Beta)
- **Environment:** Termux / Linux
- **Security:** Dotenv (Environment Variable Encryption)

 Security Features
- **Zero-Key Exposure:** Uses `.env` configuration to prevent API key leaks.
- **Production-Ready Git:** Includes `.gitignore` protocols to protect sensitive credentials.
